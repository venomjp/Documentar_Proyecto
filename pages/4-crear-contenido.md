**LibDoc** se desarrolló para facilitar la creación de documentación, por lo que se recomienda usar Markdown ```.md``` y ```html``` para escribir las plantillas que se convertirán en páginas web.

Los archivos Markdown se pueden combinar con HTML. 

Además, hay disponible una colección de código HTML predefinido como widgets que podemos copiar y reutilizar.

* [Markdown](./4_1-markdown.md) para semántica y sintaxis comunes.
* [HTML en línea](./4_2-html.md) para mejorar la semántica no compatible con Markdown.
* [Widgets](./4_3-widgets.md) para usos avanzados y específicos.
