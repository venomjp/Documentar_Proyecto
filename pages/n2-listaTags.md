La lista de tags muestra los tags definidos y el número de veces que se utilizan. Se muestra en la barra lateral. Al hacer ```clic``` en un tag, se mostrarán todas las páginas o entradas asociadas a ese tag específico, facilitando así la exploración del contenido relacionado. Los tags se definen en el front matter de cada plantilla.

{% alertAlt 'info','Nota' %}
Los <b>tags</b> se utilizan para categorizar el contenido. Es importante mantener una lista de tags consistente, no demasiado amplia ni demasiado específica ya que disponemos de la opción de búsqueda directa y difusa para encontrar contenido específico.
{% endalertAlt %}
